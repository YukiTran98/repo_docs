## Importance bar

```python
def importance_bar(score: int) -> str:
    filled = round(score / 10)
    return '█' * filled + '░' * (10 - filled)

# score=92 → '█████████░'
# score=74 → '███████░░░'
# score=48 → '████░░░░░░'
```

## Badges

Dùng inline text:
- `[ENTRY POINT]` — là entry point (main, index, app, cmd/)
- `[PUBLIC]` — exported/public API
- `[SIDE EFFECTS]` — có I/O, DB call, network, filesystem, global state

## Sorting rules

1. Files có entry point functions lên đầu
2. Trong mỗi file: entry points trước, sau đó sort by importance_score desc
3. Classes trước individual functions nếu cùng score
4. Private/internal functions (leading underscore) xuống cuối

### `command` [ENTRY POINT] [PUBLIC]

**Type:** `function`
**Importance:** 97/100 `██████████`

```python
def command(
    name: str | None = None,
    cls: type[Command] | None = None,
    **attrs: t.Any
) -> Command
```

Core decorator để biến một Python function thành CLI command. Entry point chính của toàn bộ Click API — hầu hết mọi CLI app bắt đầu từ đây.

Khi apply lên function, Click tạo một `Command` object, wrap function đó làm callback, và register các `@option`/`@argument` đã stack. Function gốc trở thành callable CLI command có thể invoke trực tiếp hoặc add vào Group.

**Used in:** Tất cả user code, `src/click/decorators.py` (group wrapper)

---

### `Context` [PUBLIC]

**Type:** `class`
**Importance:** 91/100 `█████████░`

```python
class Context:
    def __init__(
        self,
        command: Command,
        parent: Context | None = None,
        info_name: str | None = None,
        **kwargs
    )
```

Trung tâm của mọi command invocation. Lưu toàn bộ execution state: parsed params, parent/child context chain, `obj` cho shared data, metadata về command đang chạy.

Mỗi invocation tạo ra một `Context` mới. Subcommands tạo child contexts — cho phép share state up/down chain qua `ctx.parent.obj`. User access qua `@pass_context` hoặc `click.get_current_context()`.

**Used in:** `src/click/decorators.py`, `src/click/termui.py`, `src/click/testing.py`
