04 Onboarding quick start

Install & run

Step 1 — create & activate virtualenv
python -m venv .venv
# Windows:
.venv\Scripts\activate
# macOS / Linux:
source .venv/bin/activate
Step 2 — install dependencies
pip install -r requirements.txt
Step 3 — setup env vars
# tạo file .env trong thư mục agent_basic/
OPENAI_API_KEY=sk-...
NOTION_TOKEN=secret_...  # nếu dùng Notion agent
Step 4 — ingest PDF (nếu dùng Knowledge agent)
# đặt PDF vào create_vector_db/data/document.pdf
python create_vector_db/ingest_db.py
Step 5 — run
# CLI REPL:
python main.py
# Web UI (port 8000):
uvicorn web_main_agent:app --reload

Key things to know

MCP qua stdio transport. Mỗi MCP server được spawn như subprocess, giao tiếp qua stdin/stdout — không phải HTTP. AsyncExitStack quản lý vòng đời toàn bộ connections.

Main Agent là pure router. gpt-4.1 không thực thi tool trực tiếp — nó chỉ quyết định sub-agent nào được gọi dựa trên intent của user query.

MCPAgent là generic. MCPAgent(session) nhận bất kỳ MCP session nào, tự list_tools() và gọi OpenAI — không cần biết domain cụ thể.

PDF ingest là one-time setup. Chạy ingest_db.py một lần trước khi dùng knowledge agent. FAISS DB được persist tại vectorstores/pdf_faiss/.

Weather dùng NWS API (US only). get_alert cần 2-letter state code (CA, NY...) — Main Agent prompt đã có instruction convert tên đầy đủ sang code.

Web UI giữ một stack duy nhất. lifespan giữ MCP connections sống xuyên suốt app — không reconnect mỗi request. Đóng gracefully khi shutdown.

05 Project requirements
Runtime — Python 3.10+ (async/await, match syntax)
OPENAI_API_KEY — bắt buộc cho tất cả agent calls và PDF embedding
NOTION_TOKEN — bắt buộc nếu dùng Notion agent (notion_mcp_server.py)
openai-agents — OpenAI Agents SDK (Agent, Runner, function_tool)
mcp >= 0.2.0 — MCP client/server (ClientSession, FastMCP, StdioServerParameters)
langchain + faiss-cpu — PDF loading, text splitting, FAISS vector store (knowledge agent)
fastapi + uvicorn — Web UI và WebSocket endpoint (web_main_agent.py)
httpx — async HTTP cho Weather (NWS API) và Notion API
PDF file — cần đặt tại create_vector_db/data/document.pdf trước khi chạy ingest_db.py
