**Type:** `class`
**Importance:** 95/100 `█████████░`

```python
class MCPAgent:
    def __init__(self, session: ClientSession, model: str = 'gpt-4.1')
```

Generic MCP-backed agent — nhận một `ClientSession` đã được khởi tạo, không biết cụ thể domain là gì. Tự phát hiện tools qua `session.list_tools()` mỗi lần `run()` được gọi. Độc lập với domain: cùng một class được dùng cho Weather, Notion, và PDF Knowledge Base.

**Used in:** `main.py:init_main_agent()` (tạo 3 instances — `notion_agent`, `weather_agent`, `knowledge_agent`)

---

### `MCPAgent.run` [PUBLIC] [SIDE EFFECTS]

**Type:** `method`
**Importance:** 95/100 `█████████░`

```python
async def run(self, query: str) -> str
```

Core agent execution loop: (1) `session.list_tools()` → convert sang OpenAI tool format; (2) gọi `client.chat.completions.create()` với tools; (3) nếu response có `tool_calls` → extract name + args → `session.call_tool()` → collect content text; (4) feed kết quả tool lại cho model trong followup call; (5) trả `content` của followup.

Hiện chỉ xử lý `tool_calls[0]` (first tool call) — multi-tool trong một turn chưa được hỗ trợ.

**Used in:** `main.py:call_notion_agent()`, `main.py:call_weather_agent()`, `main.py:call_knowledge_agent()`

### `ingest_pdf` [ENTRY POINT] [SIDE EFFECTS]

**Type:** `function`
**Importance:** 50/100 `█████░░░░░`

```python
def ingest_pdf() -> None
```

Pipeline setup một lần duy nhất: `PyPDFLoader` load PDF → `RecursiveCharacterTextSplitter` (chunk_size=800, overlap=100) tạo chunks → `OpenAIEmbeddings(text-embedding-3-small)` embed mỗi chunk → `FAISS.from_documents()` → `db.save_local('./vectorstores/pdf_faiss')`. In `"Successfully created vector db."` khi xong.

Phụ thuộc vào `OPENAI_API_KEY` trong env và PDF tại `./create_vector_db/data/document.pdf`.

**Used in:** `create_vector_db/ingest_db.py` (`if __name__ == '__main__'`)
