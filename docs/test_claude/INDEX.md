## 🎓 Key Concepts

TypedDict defining the state schema with fields:
- `task`: Original user input

- `subtask_results`: Results from executor
- `current_index`: Progress tracker
- `final_answer`: Synthesized output
- `num_steps`: Step counter

### Workflow Routing
Uses conditional edges:
- After decomposer: Always go to executor
- After executor: Check if more subtasks → execute or synthesize
